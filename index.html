TL;DR

Awake is being built for reliable mornings and focused days.

Set mission alarms that require action, block distracting apps with Screen Time permission, run Pomodoro and countdown timers, choose custom sounds, and use wake-up checks that make it harder to fall back asleep. Awake is currently in development for iPhone.

Features

The core features planned for alarms, focus, and app blocking.

01

Mission alarm menu

Build wake-up alarms with repeat days, snooze controls, emoji labels, gentle wake-up, wake-up checks, wallpapers, and time-zone anchored ringing.

02

Wake-up missions

Dismiss alarms with math, memory match, typing, color tiles, tic tac toe, steps, shaking, QR/barcode scans, item hunts, squats, push-ups, affirmations, and verse missions.

03

Pomodoro focus

Run focused work blocks with Pomodoro presets, custom durations, task selection, focus notes, strict mode, flip start, and anti-burn-in display support.

04

Countdown and stopwatch

Use countdown presets, multi-timer support, stopwatch tracking, lap history, and focused timer modes for study, work, workouts, cooking, or timed sessions.

05

App blocking

Select distracting apps and lock them during focus or alarm enforcement. App blocking uses iOS Screen Time permission so the phone supports the accountability layer.

06

Custom sounds

Pick alarm tones, focus sounds, downloaded sounds, Spotify-style music options, or record a custom sound up to two minutes and preview it before saving.

Alarm menu

Detailed controls planned inside every wake-up alarm.

Schedule

Time, repeat, and snooze

Set the exact alarm time, choose repeat days, add a custom label and emoji, and control snooze behavior for each alarm.

Missions

Brain and body wake-ups

Add one or more wake-up tasks, including puzzle, typing, movement, scan, household item hunt, affirmation, and verse-based missions.

Sound

Sound and behavior

Choose alarm sounds, adjust volume, use custom recordings, enable gentle wake-up, and turn on a wake-up check after dismissal.

Travel

Time-zone anchor

Anchor an alarm to a selected city or time zone so travel, remote work, and time-zone changes do not break the intended alarm schedule.

Lock

Alarm lock controls

Planned alarm lock settings include app protection, switch-off protection, and Screen Time-based blocking for stronger wake-up accountability.

Visual

Wallpaper and motivation

Customize the alarm screen with wallpaper and optional daily motivation so the wake-up screen feels personal, not generic.
